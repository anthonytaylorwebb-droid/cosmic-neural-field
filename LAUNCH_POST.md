# Launch Post

Built a new local-first macOS app called **Signal Field**.

It turns real system activity into a living neural/cosmic interface:

- CPU
- memory
- thermal state
- battery pressure
- disk I/O
- network traffic
- top processes

Instead of dashboards and charts, it renders the machine like a glowing field of particle-cluster nodes, flowing influence paths, and process constellations.

Tech:

- React
- TypeScript
- Vite
- Canvas 2D
- Electron

Runs fully local on macOS. No account required.
